- (event) Versions verified against npm registry on 2026-07-10, not recalled: next 16.2.10, payload 3.85.2, react 19.2.7, tailwindcss 4.3.2, typescript 7.0.2, pnpm 11.11.0, node 24.18.0 LTS.
- (decision) Payload 3.85.2 peer-requires next '>=16.2.6 <17'. Next 16.2.10 satisfies it. Pin both; a Next minor bump outside that range breaks the admin route.
- (decision) Supabase transaction pooler (6543) does not support prepared statements; DDL/migrations run over the direct connection or session pooler (5432). Two connection strings, not one.
- (decision) Payload push mode is dev-only. Production schema changes go through committed Drizzle migrations. Mixing push and migrate corrupts state -- Payload docs are explicit.
- (decision) Media on Supabase Storage via @payloadcms/storage-s3 against Supabase's S3-compatible endpoint, so content and assets share one provider.
- (decision) Structured data is derived from CMS documents at render, never hand-authored in templates -- FR-11/FR-12 demand it stay in sync with openingHours and areaServed.
- (decision) Rejected @payloadcms/plugin-seo as the source of truth for meta. The 22 hand-written titles/descriptions are seeded as data (FR-10); the plugin would invite regeneration.
- (decision) Lenis removed entirely after client feedback ('scrolls like a slowpoke'): scrolling is native, scrollbar branded via CSS scrollbar-color + -webkit pseudo-elements on tokens. Motion = reveals/hovers/Embla only.
- (event) Design reference v2 (light hero, arch photo mask, floating trust card) received 2026-07-10; same invented facts as v1 (24/7, wrong phone, 15+/5000+, Warszawa i okolice) plus misspelled UNIK and three unserviced brands in the logo wall. Visual language adopted, facts kept verified. docs/design/reference-v2-analysis.md
- (decision) AD-17 added: brands.authorized and brands.distributor are independent booleans, both default false until client answers OQ-11. Sales path = 4th Service seed + leads.topic value + distributor flag; no catalog/e-commerce in v1 (OQ-10 recommendation).
- (decision) Surface token 'mist' #EAF1EC added (usage surface-under-dark-text, measured vs muted 5.99:1). Zebra banding on homepage per section-rhythm pattern. brand-green on mist = 2.98:1 -- banned by test; step numerals use link-green.
